# cognifyz-task1_rating_prediction
# Restaurant Rating Prediction using Machine Learning

## Project Overview

This project was developed as part of the Cognifyz Technologies Machine Learning Internship.

The objective of this project is to build a Machine Learning model that predicts the aggregate rating of a restaurant based on various features such as the number of votes, price range, and average cost for two people.

The project follows a complete Machine Learning workflow including data loading, preprocessing, model training, evaluation, and visualization of results.

## Dataset

The dataset contains information about restaurants, including:

* Restaurant Name
* City
* Cuisines
* Average Cost for Two
* Price Range
* Votes
* Aggregate Rating
* Latitude and Longitude

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Machine Learning Algorithm

Linear Regression was used to predict restaurant ratings because the target variable (Aggregate Rating) is a continuous numerical value.

## Project Workflow

1. Load the restaurant dataset.
2. Explore and understand the dataset.
3. Check and handle missing values.
4. Select relevant features:

   * Votes
   * Price Range
   * Average Cost for Two
5. Split the dataset into training and testing sets.
6. Train a Linear Regression model.
7. Predict restaurant ratings.
8. Evaluate model performance using:

   * Mean Squared Error (MSE)
   * R² Score
9. Visualize Actual vs Predicted Ratings.

## Results

The model successfully predicts restaurant ratings based on restaurant characteristics and provides insights into how factors such as votes, pricing, and cost influence customer ratings.
